## 🌐 النشر على Vercel

### خطوات النشر:

1. **Push to GitHub:**
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/subtract.git
git push -u origin main
```

2. **Deploy to Vercel:**
   - روح https://vercel.com
   - Import the GitHub repo
   - Deploy

3. **Add Vercel Domain to Firebase:**
   - خد الـ domain من Vercel (مثلاً: `subtract-xyz.vercel.app`)
   - روح Firebase Console → Authentication → Settings → Authorized domains
   - Add domain → الصق الـ Vercel domain
   - Save

**✅ خلاص! شغال على Vercel**
